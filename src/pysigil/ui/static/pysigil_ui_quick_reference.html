pysigil quick reference

Key controls & pill legend

A fast, visual guide to the pills and controls you see in the preferences window. Scopes are ordered left → right from highest to lowest priority.

Getting oriented

  • Provider selects which package you’re setting values for. It remembers your last pick.
  • Project shows the active project path (read‑only).
  • Compact toggles a tighter layout for dense pages.

Working with fields

  • Tooltips: hover the i for field info; hover a pill to see details about that scope.
  • Editing: click Edit… or click a pill for that scope to edit directly.
  • Effective chip shows the value currently applied and where it comes from.
